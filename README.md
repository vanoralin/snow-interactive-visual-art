# ❄️ Snowfall Body Outline Interactive Art

โปรเจกต์งานศิลปะเชิงปฏิสัมพันธ์แบบเรียลไทม์ (Interactive Visual Art) ที่ผสมผสานเว็บแอปพลิเคชันและปัญญาประดิษฐ์ (AI) เพื่อตรวจจับโครงร่างร่างกายของผู้ใช้งานผ่านกล้องเว็บแคม จากนั้นจำลองทิศทางการตกของหิมะ ให้หิมะตกลงมาชน ไหลสไลด์ลู่ไปตามศีรษะและบ่าของคุณ แล้วร่วงหล่นลงสู่พื้นดินอย่างสมจริง

---

## ✨ คุณสมบัติเด่น (Features)
- **AI Body Segmentation**: ใช้โมเดล **MediaPipe Image Segmenter** บน Python ที่ประมวลผลได้อย่างรวดเร็วและแม่นยำสูง เพื่อตัดขอบเขตของร่างกายมนุษย์ในแบบเรียลไทม์
- **Real-Time WebSocket**: สื่อสารอัปโหลดเฟรมภาพและพิกัดขอบเขตระหว่างเบราว์เซอร์ (Frontend) และ Python Server (Backend) ผ่าน WebSockets ภายในเครื่องอย่างลื่นไหลไร้รอยต่อ
- **Natural Snow Physics (อนุรักษ์จำนวนเม็ดหิมะ 100%)**:
  - เม็ดหิมะที่ไหลตกจากตัวมีจำนวนและขนาดเท่ากับหิมะที่ตกลงมาชนจริง ๆ (ไม่มีการสร้างเม็ดหิมะปลอมขึ้นมาบนร่างกาย)
  - หิมะตกกระทบแล้วจะไหลสไลด์ลงด้านล่างตามความลาดชันของศีรษะและบ่าทันที
  - สำหรับแนวระนาบตรง (เช่น กลางศีรษะ) เม็ดหิมะจะทำการสุ่มเลี้ยวซ้ายหรือขวาเพื่อไหลลงต่อทันที ทำให้ไม่มีหิมะเกาะติดค้างอยู่ถาวร
- **Smart Fallback**: หากไม่ได้เปิดกล้องหรือ Python Server ปิดตัวอยู่ ระบบจะแสดงเงาจำลองที่ขยับตามเมาส์/นิ้วสัมผัส เพื่อให้สามารถเล่นเกาะหิมะได้ทันทีโดยไม่แครช
- **Clean Camera View**: แสดงภาพวิดีโอจากกล้องด้วยสีสันธรรมชาติ คมชัด 100% ไม่มีแผ่นฟิลเตอร์มืดทับซ้อน

## 🏗️ โครงสร้างของระบบ (Architecture)

```
[ Web Browser (Frontend) ]
      │
      │ 1. ส่งเฟรมกล้องขนาด 320x240 (JPEG Binary Blob) ทุกๆ 16ms
      ▼
[ Python Server (Backend: ws://localhost:8765) ] ──► ประมวลผลโมเดล MediaPipe Tasks AI
      │
      │ 2. ส่งข้อมูลพิกัดขอบบนบ่า/หัว (JSON) กลับมา
      ▼
[ HTML5 Canvas (Render) ] ──► จำลองฟิสิกส์หิมะสไลด์หล่นตามขอบ
```

---

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

### ฝั่งผู้ใช้งาน (Frontend)
- **Vite** (เครื่องมือจัดเตรียมพัฒนาเว็บความเร็วสูง)
- **pnpm** (เครื่องมือจัดการแพ็กเกจ Node.js ประสิทธิภาพสูง)
- **HTML5 Canvas API** (ใช้ในการวาดเม็ดหิมะและคำนวณฟิสิกส์ลื่นไหล)

### ฝั่งเซิร์ฟเวอร์ AI (Backend)
- **Python 3.10+**
- **MediaPipe Tasks API** (ใช้ลอจิกวิเคราะห์คัดแยกตัวคนออกจากพื้นหลัง)
- **OpenCV** (ใช้สำหรับจัดสรรข้อมูลเฟรมกล้องแบบไบนารี)
- **Websockets** (จัดการการส่งผ่านเฟรมภาพและพิกัดระยะหน่วงต่ำ)

## 🚀 วิธีการติดตั้งและเริ่มใช้งาน (Setup & Installation)

### 1. การตั้งค่าฝั่งเซิร์ฟเวอร์ AI (Python Backend)

1. เข้าไปที่โฟลเดอร์ของเซิร์ฟเวอร์:
   ```bash
   cd backend
   ```
2. ติดตั้งไลบรารีที่จำเป็นผ่าน pip:
   ```bash
   pip install -r requirements.txt
   ```
3. สตาร์ทเซิร์ฟเวอร์หลังบ้าน:
   ```bash
   python main.py
   ```
   *(เมื่อเปิดรันครั้งแรก ระบบจะดาวน์โหลดไฟล์โมเดล AI `selfie_segmenter.tflite` จาก Google CDN มาเก็บไว้ในเครื่องโดยอัตโนมัติ และจะขึ้นข้อความ `WebSocket AI detection server started on ws://localhost:8765`)*

---

### 2. การตั้งค่าฝั่งหน้าเว็บ (Node.js Frontend)

1. เปิด Terminal ใหม่ในโฟลเดอร์หลักของโปรเจกต์
2. ติดตั้ง Dependencies ด้วย pnpm:
   ```bash
   pnpm install
   ```
3. รันเซิร์ฟเวอร์จำลองการพัฒนา:
   ```bash
   pnpm run dev
   ```
4. เปิดเบราว์เซอร์ไปที่ลิงก์ที่แสดงบนหน้าจอ (ปกติคือ [http://localhost:5173/](http://localhost:5173/))
5. กดปุ่ม **"เปิดกล้อง AI"** บนหน้าจอเพื่อเริ่มเล่น!
